# МО — СИЛА · Генератор документов практики

Сычёв Иван

Скрипт автоматически генерирует четыре документа практики для каждого студента на основе Excel-таблицы: **извещение**, **отзыв**, **индивидуальное задание** и **отчёт**.

## Установка

### 1. Клонировать репозиторий

```bash
git clone https://github.com/your-username/practice-doc-generator.git
cd practice-doc-generator
```

### 2. Создать виртуальное окружение

```bash
python3 -m venv .venv
source .venv/bin/activate        # macOS / Linux
.venv\Scripts\activate           # Windows
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Запустить

```bash
python main.py
```

### Поле 26 — содержание отчёта

Это основной текст отчёта студента. Чтобы разбить текст на **абзацы с красной строкой**, используй метку `(отступ)` между абзацами прямо в ячейке Excel:

```
Первый абзац текста.(отступ)Второй абзац текста.(отступ)Третий абзац текста.
```

Каждый фрагмент после `(отступ)` будет оформлен как отдельный абзац с:
- отступом первой строки 1.25 см
- межстрочным интервалом 1.5
- выравниванием по ширине
- шрифтом Times New Roman 14pt

## Как пользоваться программой

### Шаг 1 — загрузить Excel
Нажми **Обзор...** и выбери заполненный `.xlsx` файл.  
Программа покажет сколько студентов найдено.

### Шаг 2 — выбрать студентов
- **Все студенты** — сгенерирует документы для всех
- **Один студент** — введи номер строки (3 = первый студент, 4 = второй и т.д.)
- **Диапазон строк** — например с 3 по 7 — сгенерирует для пяти студентов

### Шаг 3 — выбрать документы
Отметь галочками только те документы которые нужно сгенерировать.

### Шаг 4 — папка сохранения
По умолчанию документы сохраняются рядом с Excel-файлом.  
Можно изменить через кнопку **Изменить**.

### Генерация
Нажми **Сгенерировать документы**.  
Для каждого студента создаётся отдельная папка с его ФИО, внутри — выбранные документы.

```
📁 output/
├── 📁 Иванов_Иван_Иванович/
│   ├── извещение.docx
│   ├── отзыв.docx
│   ├── индзадание.docx
│   └── отчет.docx
├── 📁 Петров_Пётр_Петрович/
│   └── ...
```

## Шаблоны

Шаблоны хранятся в папке `templates/`. В них расставлены метки вида `{{N}}` где N — номер структурного элемента из таблицы выше. При генерации скрипт заменяет метки на данные из Excel.

> Шаблоны не входят в репозиторий — добавь их вручную в папку `templates/`.
